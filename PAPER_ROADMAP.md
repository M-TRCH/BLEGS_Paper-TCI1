## ส่วนที่ 2 -- เนื้อหาบทความ (ตามโครงสร้างบังคับของวารสาร)

### 1. บทนำ
- [ ] ความสำคัญและที่มาของปัญหา: ข้อจำกัดด้านค่าแรงบิดต่อน้ำหนัก (Torque-to-Weight Ratio) ของหุ่นยนต์สี่ขา
- [ ] ปัญหาเมื่อเลือกมอเตอร์ใหญ่ (TWR ลดลง) vs ลดน้ำหนักโครงสร้าง (ความแม่นยำลดลงจากระยะคลอน)
- [ ] วรรณกรรมที่เกี่ยวข้อง: กลไก Five-Bar Linkage, ระบบขับเคลื่อน BLDC/FOC, การติดตามตำแหน่งด้วย Vision
- [ ] ช่องว่างงานวิจัยและจุดเด่นของงาน: แนวทาง Data-Driven Compensation โดยใช้ Encoder + ArUco Vision
- [ ] วัตถุประสงค์ของงานวิจัย
- [ ] สมมติฐาน (ถ้ามี)

### 2. วัสดุ อุปกรณ์ และวิธีการวิจัย
#### 2.1 แพลตฟอร์มหุ่นยนต์ BLEGS
- [ ] การออกแบบเชิงกล -- โครงสร้าง 2-DOF Separated Five-Bar Linkage ต่อขา
- [ ] สเปคมอเตอร์ BLDC, อัตราทด 8:1, Absolute Encoder (AS5047P 14-bit)
- [ ] ไมโครคอนโทรลเลอร์ STM32G431CBU6 (170 MHz ARM Cortex-M4)
- [ ] การควบคุม FOC/SVPWM, PID Position Control
- [ ] สื่อสาร Binary Protocol v1.2 (UART 921,600 baud)
- [ ] ขนาดลิงก์: L_AC = L_BD = 105 มม., L_CE = L_DE = 145 มม., ระยะห่างมอเตอร์ 85 มม.

#### 2.2 การทดลองจริง
- [ ] ขั้นตอนการสั่งเดินด้วย Gait Trajectory (CSV setpoints)
- [ ] การบันทึก Log มุมมอเตอร์ (Setpoint vs Actual) จาก Encoder
- [ ] เงื่อนไขการทดลอง: ความเร็วเดิน, จำนวนรอบ, อัตราการส่งคำสั่ง (Hz)

#### 2.3 การวัดตำแหน่งเชิงภาพ (Vision-based Position Estimation)
- [ ] ติด ArUco Marker บนข้อต่อ (ID 1=A, 2=B, 3=C, 4=D)
- [ ] Camera Calibration -- พารามิเตอร์กล้อง Sigma 35mm APSC, 1080p60
- [ ] คำนวณตำแหน่งปลายขา (E) จาก Circle Intersection ใน Pixel Space
- [ ] แปลง Pixel Coordinate เป็น Robot Frame (มม.) อ้างอิงจุด A, B

#### 2.4 แนวทางชดเชยความคลาดเคลื่อน (Data-Driven Compensation)
- [ ] เปรียบเทียบตำแหน่งจาก Encoder (Forward Kinematics) กับ Vision (ArUco)
- [ ] สร้างแบบจำลองชดเชย (Corrected Position Model)
- [ ] วิธีการประเมินผลก่อนและหลังชดเชย

### 3. ผลการทดลองและอภิปรายผลการทดลอง
#### 3.1 การเปรียบเทียบ Trajectory
- [ ] กราฟเปรียบเทียบเส้นทางปลายขา: Target vs Encoder vs Vision (2D Plot)
- [ ] กราฟเปรียบเทียบมุมมอเตอร์: Target vs Actual ของทุกขา (8 มอเตอร์)

#### 3.2 การวิเคราะห์ค่าผิดพลาด
- [ ] ตาราง RMSE ของมุมมอเตอร์ (แยกตามขาและแกน)
- [ ] ตาราง RMSE ของตำแหน่งปลายขา (แกน X, Y, รวม)
- [ ] การวิเคราะห์ความถี่หลัก (Dominant Frequency) ด้วย FFT

#### 3.3 ผลการชดเชย
- [ ] กราฟ Trajectory ก่อนและหลังชดเชยซ้อนทับกัน
- [ ] ตารางเปรียบเทียบ RMSE ก่อน/หลัง
- [ ] สรุปเชิงสถิติ: ค่าเฉลี่ย, ส่วนเบี่ยงเบนมาตรฐาน, ค่าผิดพลาดสูงสุด

#### 3.4 การวิเคราะห์ Torque-to-Weight Ratio
- [ ] ตารางเปรียบเทียบ TWR กับงานวิจัยอื่น
- [ ] กราฟ Scatter: น้ำหนัก vs TWR ของหุ่นยนต์สี่ขาต่าง ๆ
- [ ] กราฟ Bar Chart: TWR ของ BLEGS เทียบกับหุ่นยนต์อ้างอิง

### 4. อภิปรายผลและสรุป
- [ ] สาเหตุของความคลาดเคลื่อนระหว่าง Encoder กับ Vision
- [ ] ผลกระทบจากระยะคลอน (Joint Slop) และแรงดัดในลิงก์ (Link Deflection)
- [ ] ข้อจำกัดของวิธีชดเชย
- [ ] เปรียบเทียบผลลัพธ์กับงานวิจัยที่เกี่ยวข้อง
- [ ] สมดุลระหว่างประสิทธิภาพเชิงกลและความแม่นยำเชิงตำแหน่ง
- [ ] สรุปผลการวิจัยตามวัตถุประสงค์
- [ ] ข้อเสนอแนะและแนวทางการพัฒนาในอนาคต
