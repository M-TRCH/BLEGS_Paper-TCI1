# แผนงานบทความวิจัย -- TCI Tier 1 (KMUTNB Int. J. Appl. Sci. Technol.)

## 1. การเตรียมต้นฉบับ

- [ ] ชื่อเรื่องสองภาษา (ไทย/อังกฤษ) -- ตรวจสอบและยืนยัน
- [ ] บทคัดย่อภาษาอังกฤษ -- ไม่เกิน 250 คำ ครอบคลุม วัตถุประสงค์ วิธีการ ผลลัพธ์ สรุป
- [ ] บทคัดย่อภาษาไทย -- ไม่เกิน 250 คำ ตรวจสอบโดยเจ้าของภาษา
- [ ] คำสำคัญ 3-5 คำ ทั้งสองภาษา

## 2. บทนำ

- [ ] คำชี้แจงปัญหา: ข้อจำกัดด้าน Torque-to-Weight Ratio ของหุ่นยนต์สี่ขา
- [ ] การทบทวนวรรณกรรม: กลไก Five-Bar Linkage, ระบบขับเคลื่อน BLDC, การติดตามตำแหน่งด้วย Vision
- [ ] ช่องว่างงานวิจัยและจุดเด่นของงาน
- [ ] วัตถุประสงค์ของงานวิจัย

## 3. วิธีการวิจัย

### 3.1 แพลตฟอร์มหุ่นยนต์
- [ ] การออกแบบเชิงกลของ BLEGS -- โครงสร้างและจลนศาสตร์ 2-DOF Separated Five-Bar Linkage
- [ ] สเปคมอเตอร์ BLDC, อัตราทด 8:1, Absolute Encoder (AS5047P)
- [ ] สื่อสารผ่าน Binary Protocol v1.2 (UART 921,600 baud)
- [ ] ขนาดลิงก์: L_AC = L_BD = 105 mm, L_CE = L_DE = 145 mm

### 3.2 การทดลองจริง (Real Hardware)
- [ ] ขั้นตอนการทดลอง -- การสั่งเดินด้วย Gait Trajectory (CSV setpoints)
- [ ] การบันทึก Log มุมมอเตอร์ (Setpoint vs Actual) จาก Encoder
- [ ] เงื่อนไขการทดลอง -- ความเร็วเดิน, โหลด, จำนวนรอบ

### 3.3 การวัดตำแหน่งเชิงภาพ (Vision-based Position Estimation)
- [ ] ขั้นตอน ArUco Marker Detection -- ติด Marker บนข้อต่อ (ID 1-4)
- [ ] Camera Calibration -- พารามิเตอร์กล้อง (Sigma 35mm APSC, 1080p60)
- [ ] คำนวณตำแหน่งปลายขา (End-Effector E) จาก Circle Intersection ใน Pixel Space
- [ ] แปลง Pixel Coordinate เป็น Robot Frame (mm) โดยอ้างอิงจุด A, B

### 3.4 แนวทางชดเชยความคลาดเคลื่อน (Data-Driven Compensation)
- [ ] เปรียบเทียบตำแหน่งจาก Encoder (Forward Kinematics) กับ Vision (ArUco)
- [ ] สร้างแบบจำลองชดเชย (Corrected Position Model)
- [ ] ประเมินผลก่อนและหลังชดเชย

## 4. ผลการทดลอง

### 4.1 การเปรียบเทียบ Trajectory
- [ ] กราฟเปรียบเทียบเส้นทางปลายขา: Target vs Encoder vs Vision (2D Plot)
- [ ] กราฟเปรียบเทียบมุมมอเตอร์: Target vs Actual ของทุกขา (8 มอเตอร์)

### 4.2 การวิเคราะห์ค่าผิดพลาด
- [ ] ตาราง RMSE ของมุมมอเตอร์ (แยกตามขาและแกน)
- [ ] ตาราง RMSE ของตำแหน่งปลายขา (แกน X, Y, รวม)
- [ ] การวิเคราะห์ความถี่หลัก (Dominant Frequency) ด้วย FFT

### 4.3 ผลการชดเชย
- [ ] กราฟ Trajectory ก่อนและหลังชดเชยซ้อนทับกัน
- [ ] ตารางเปรียบเทียบ RMSE ก่อน/หลัง
- [ ] สรุปเชิงสถิติ -- ค่าเฉลี่ย, ส่วนเบี่ยงเบนมาตรฐาน, ค่าผิดพลาดสูงสุด

### 4.4 การวิเคราะห์ Torque-to-Weight Ratio
